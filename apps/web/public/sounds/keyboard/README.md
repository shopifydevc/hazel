# Keyboard Sound Assets

This directory contains mechanical keyboard sound files for the typing experience.

## Sound Types

- `cherry-mx-blue.mp3` - Cherry MX Blue switch sound
- `cherry-mx-brown.mp3` - Cherry MX Brown switch sound
- `cherry-mx-red.mp3` - Cherry MX Red switch sound
- `topre.mp3` - Topre switch sound
- `alps-blue.mp3` - Alps Blue switch sound

## Adding Custom Sounds

To add your own keyboard sounds:

1. Record or download mechanical keyboard sound samples
2. Convert to MP3 format (recommended for web compatibility)
3. Keep file sizes small (< 50KB recommended)
4. Name files descriptively
5. Add them to this directory

## Sound Requirements

- Format: MP3
- Duration: 50-200ms recommended
- Volume: Normalized to prevent clipping
- Quality: 44.1kHz sample rate recommended
